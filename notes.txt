HOW DOES VIRTUAL MAPPINGS WORK HERE?

- Kernel:
    - Process address space start (virt) + size
    - KPGDir

- User process:
    - UPGDir

- If the user process passes a pointer to the kernel process, then kernel process calculates the pointer in the kernel's address space by calculating offsets.
