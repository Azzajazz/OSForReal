typedef enum {
    PT_BOOT,
    PT_GENERIC,
} Partition_Type;

typedef struct {
    uint8_t magic[4]; // Contains "PART"
    uint8_t checksum; // All bytes must add to 0 mod 0x100 (including header and all partition entries).
    uint8_t version;
    uint16_t partition_count;
} Partition_Header;

typedef struct {
    Partition_Type type;
    uint32_t sector_start;
    uint32_t sector_count;
} Partition_Entry;





Disk partition header:
- Magic bytes
- Checksum  (ala ACPI RSDT)
- Number of paritions.
- Array of partition entries.
    - Partition type.
    - Partition start sector.
    - Partition sector count.

In order to check for a boot parition disk:
- Check for magic bytes
- Verify using checksum
- Walk partitions and see if there's a boot partition. For now, we can just assume there's one of them.

- This needs to be done when enumerating devices. One of the storage devices will have a boot partition.
